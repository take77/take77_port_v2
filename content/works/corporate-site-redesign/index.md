## プロジェクト概要

製造業クライアントの10年以上更新されていないコーポレートサイトを全面リニューアルしました。旧サイトはFlashを使用した箇所も残っており、スマートフォン非対応・表示速度の低下・CMSの老朽化が深刻な問題となっていました。

新サイトはNext.js（App Router）とmicroCMSを採用し、マーケティング担当者がノーコードでコンテンツを更新できる体制を整えました。

## 課題

- **パフォーマンス**: Lighthouseスコア32（モバイル）→ 事業機会の損失
- **更新工数**: 都度エンジニアへの依頼が必要で、情報が陳腐化していた
- **SEO**: 構造化データなし、Core Web Vitals全項目で失格
- **スマートフォン対応**: レイアウトが崩れており、モバイルユーザーが離脱

## アプローチ

### 技術選定

| 項目 | 選定 | 理由 |
|------|------|------|
| フレームワーク | Next.js 14（App Router） | SSG + ISRで高速配信、TypeScriptとの親和性 |
| CMS | microCMS | 国産で日本語サポートが充実、APIファースト設計 |
| スタイリング | Tailwind CSS | 一貫したデザインシステムの構築が容易 |
| ホスティング | Vercel | Next.jsとの最適化、プレビュー環境が充実 |

### 実装プロセス

1. **情報設計**: 既存コンテンツの棚卸し → サイトマップの再設計
2. **デザイン**: Figmaでワイヤーフレーム → プロトタイプ → クライアント承認
3. **開発**: コンポーネント設計 → API連携 → パフォーマンス最適化
4. **移行**: 旧URLからの301リダイレクト設定 → SEO引き継ぎ

## 成果

- **Lighthouseスコア**: モバイル32 → **94**（195%改善）
- **LCP**: 8.2秒 → **1.4秒**
- **月間オーガニック流入**: 約2倍（6ヶ月後）
- **コンテンツ更新工数**: エンジニア依頼なし → マーケ担当者が自走
- **問い合わせ数**: 前年同期比 **+40%**
